# Bajiru Speaker Recognition

Identify whether regular or low voice Bajiru is speaking

There are currently 2 implementations of this project:

- [Python (prototyping)](./python/README.md)
- [C++ (production and integration)](./cpp_impl/README.md)

## Setup

Prerequisites:

- [`uv`](https://docs.astral.sh/uv/)

Installations:

```bash
uv sync
prek install
```
